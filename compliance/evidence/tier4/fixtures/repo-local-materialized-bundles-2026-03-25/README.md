# Preserved Tier 4 Bundles

Place normalized, independently generated interop bundles here.

A preserved Tier 4 bundle must contain at minimum:
- `manifest.yaml`
- `mapping.yaml`
- `execution.log`
- `peer-profile.yaml`
- `peer-artifacts/`
- `reproduction.md`
- peer identity and runtime/container provenance in the manifest
- scenario outcomes and exact contract/profile binding
