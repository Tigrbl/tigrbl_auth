
# Claim Plane

The claim plane records:
- current repository certification posture,
- tier definitions,
- promotion rules,
- declared current claims,
- and whether governance installation is complete.

Tier definitions:
- Tier 0 — internal / experimental
- Tier 1 — declared
- Tier 2 — implemented and internally asserted
- Tier 3 — evidence-backed and release-gated
- Tier 4 — independently peer-supported
